# Introduction to Programming

## Apa itu bahasa pemrograman

Bahasa pemrograman adalah sistem notasi yang digunakan untuk menulis program komputer. Bahasa ini menyediakan sarana bagi programmer untuk memberikan instruksi kepada komputer dalam bentuk yang dapat dipahami dan dieksekusi oleh mesin.

## Kegunaan

1. Automasi Tugas: Membantu dalam mengotomatisasi tugas-tugas yang berulang dan kompleks.
   Pengembangan Aplikasi: Digunakan untuk membuat aplikasi perangkat lunak yang menjalankan berbagai fungsi.
2. Analisis Data: Digunakan dalam bidang data science untuk mengolah dan menganalisis data.
3. Pengembangan Web: Membuat dan mengelola situs web dan aplikasi web.
4. Pengembangan Sistem: Membuat sistem operasi, driver perangkat keras, dan software lain yang berinteraksi langsung dengan perangkat keras.

## Kesalahan dalam belajar pemrograman bagi pemula

1. Tidak belajar fundamental, langsung lompat ke project
2. Skip bagian yang belum dipahami
3. Mempelajari banyak bahasa sekaligus
4. Tidak konsisten
5. Berpikir untuk membuat sesuatu yang besar diawal

## Cara Belajar yang benar

1. Mulai dengan mempelajari Fundamental pemrograman
   - Basic Concept: dapat menjelaskan tiap syntax yang ditulis
   - Data Structure: dapat memahami sifat dari tiap tipe data, dan menyusun data sesuai dengan struktur yang diperlukan
   - Algoritma: dapat menuliskan code yang tepat untuk menyelesaikan suatu kasus
2. Konsisten belajar setiap hari
3. Jangan skip bagian yang belum benar-benar dipahami
4. Gabung komunitas, cari teman, cari mentor
